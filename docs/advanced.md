# Advanced Settings

These settings are for fine control.

## Mesh Settings

### Weld Distance
Merges very close vertices.

### Auto Fix Normals
Fixes shading issues automatically.

### Recalculate Outside Normals
Manual fix for broken normals.

## UV Settings

### Smart UV Angle
Controls UV splitting.

### UV Padding
Spacing between islands.

## Bake Settings

### Bake Samples
Quality of baking.

### Bake Margin
Padding between UV islands.

### JPG Quality
Compression level.

### Bake Normal
Enables normal map baking.

### Bake Occlusion
Enables AO map baking.

## Memory Safety

### Safe Memory Bake
Prevents crashes on heavy meshes.

### Force CPU Baking
Useful on low VRAM systems.

## Tip

Leave advanced settings at default unless needed.
