## Quando contatti il supporto

Includi:

- versione Blender;
- versione ScanReady;
- screenshot del problema;
- breve descrizione del problema;
- informazioni GPU / CPU se il problema e legato alle performance.

Questo aiuta a individuare i problemi piu rapidamente.
