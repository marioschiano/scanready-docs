# Riferimento impostazioni

Questa pagina riassume le impostazioni principali di ScanReady.

## Step 1

| Impostazione | Effetto | Azione consigliata |
| --- | --- | --- |
| Final Faces | cambia il target lowpoly | Create Lowpoly Preview |
| Optimize / Reduce | cambia la riduzione | Create Lowpoly Preview |
| Adaptive Preset | cambia i pesi di riduzione | Create Lowpoly Preview |
| Show Adaptive Weights | solo visualizzazione | nessun ricalcolo obbligatorio |

## Step 2

| Impostazione | Effetto | Azione consigliata |
| --- | --- | --- |
| Smart UV Preset | cambia unwrap | Generate UVs se UV non esiste, altrimenti Bake Textures |
| Smart UV Angle | cambia unwrap | Generate UVs o Bake Textures |
| UV Padding | cambia spazio tra isole | Generate UVs o Bake Textures |
| Cage Extrusion | cambia cattura bake | Bake Textures |
| Show Cage | solo visualizzazione | nessun ricalcolo obbligatorio |

## Step 3

| Impostazione | Effetto | Azione consigliata |
| --- | --- | --- |
| Texture Preset | cambia risoluzione | Bake Textures |
| Bake Materials | cambia materiale/UV split | Bake Textures |
| Bake Base Color | abilita/disabilita mappa | Bake Textures |
| Bake Normal | abilita/disabilita mappa | Bake Textures |
| Bake Roughness | abilita/disabilita mappa | Bake Textures |
| Bake Occlusion | abilita/disabilita mappa | Bake Textures |

## Advanced

| Impostazione | Effetto | Azione consigliata |
| --- | --- | --- |
| Pre-Decimate Merge | cambia mesh preview | Create Lowpoly Preview |
| Auto Fix Normals | cambia preparazione sorgente | Create Lowpoly Preview |
| Auto Clean Scan Debris | cambia pulizia mesh | Create Lowpoly Preview |
| Smart UV Angle | cambia unwrap | Generate UVs o Bake Textures |
| Bake Samples | cambia qualita bake | Bake Textures |
| JPG Quality | cambia salvataggio texture | Bake Textures |
