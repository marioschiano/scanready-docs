# Changelog

## Versione 1.0.0

Prima release pubblica di ScanReady come Blender Extension.

### Principali funzioni

- One Click Bake.
- Preview lowpoly.
- Adaptive Reduce.
- Preset Adaptive Reduce: Balanced, Preserve Details, Flat Surfaces.
- Visualizzazione Adaptive Weights.
- Smart UV Project.
- Cage preview e Auto Cage Extrusion.
- Bake Base Color, Normal, Roughness e Occlusion.
- Cache per evitare ricalcoli inutili.
- Workflow Status con suggerimenti sul prossimo step.
