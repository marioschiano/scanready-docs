# One Click Bake

The One Click Bake is the core feature of ScanReady.

It runs the full pipeline automatically.

## What it does

- Creates a lowpoly version
- Generates UVs
- Builds a cage
- Bakes textures
- Creates final optimized asset

## When to use it

Use One Click when:

- You want speed
- You don’t need manual control
- You are processing standard scans

## Tip

Start with One Click.  
Only use manual steps if needed.
