# Step 3 – Bake / Output

This step bakes textures and creates the final asset.

## Settings

### Texture Preset
Resolution of the output textures.

### Bake Materials
Number of texture sets.

### Bake Base Color
Exports the color texture.

### Save Images
Automatically saves the baked textures.

### Image Format
Available formats:

- JPG
- PNG
- TIFF

### Output Folder
Defines where textures are saved.

## Action

Click **Bake Textures** to generate the final result.
