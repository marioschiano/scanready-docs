# Step 3 – Bake / Output

This step bakes textures and creates the final asset.

## Settings

### Texture Preset
Resolution (2K, 4K, etc.)

### Bake Materials
Number of texture sets.

### Bake Base Color
Exports color texture.

### Save Images
Automatically saves textures.

### Image Format
JPG / PNG

## Output

Defines where textures are saved.

## Action

Click **Bake Textures** to generate the final result.
