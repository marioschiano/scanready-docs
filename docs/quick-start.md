# Quick Start

Get a game-ready asset in seconds using the One Click workflow.

## Steps

1. Select your high-poly scan.
2. Open the **ScanReady** panel.
3. Click **ONE CLICK BAKE**.
4. Wait for the final optimized asset.

## What happens automatically

- Mesh reduction
- Smart UV generation
- Auto cage creation
- Texture baking
- Final asset creation

This is the fastest way to process most scans.
