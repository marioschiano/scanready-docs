# Installation

ScanReady works with Blender 4.x.

## Download

Download the addon as a `.zip` file.

> Do not extract the zip file.

---

## Install the Addon

1. Open Blender  
2. Go to **Edit → Preferences**  
3. Open the **Add-ons** tab  
4. Click **Install** (top right)  
5. Select the ScanReady `.zip` file  
6. Click **Install Add-on**

---

## Enable the Addon

1. Search for **ScanReady** in the Add-ons list  
2. Enable the checkbox  
3. Click **Save Preferences**

---

## Access the Panel

- Open the **3D Viewport**
- Press **N**
- Select the **ScanReady** tab

---

## Tip

If you don’t see the panel:

- Make sure the addon is enabled
- Restart Blender
