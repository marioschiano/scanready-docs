# Installation

## Install the Addon

1. Open Blender
2. Go to **Edit → Preferences → Add-ons**
3. Click **Install**
4. Select the ScanReady `.zip` file
5. Enable the addon

## Access the Panel

- Open the **3D Viewport**
- Press **N**
- Go to the **ScanReady** tab

## Requirements

- Blender 4.x recommended
