# Step 1 – Preview / Reduce

This step creates a lowpoly version of your scan.

## Settings

### Final Faces
Target polygon count of the final mesh.

### Optimize / Reduce
Controls how aggressive the reduction is.

### Reduction %
Shows how much geometry is removed.

## View Options

- Show Wireframe → inspect topology
- Show Checker → check UV density
- Use Texture View → preview textures

## Action

Click **Create Lowpoly Preview** to generate the optimized mesh.
