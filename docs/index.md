## 🚀 What is ScanReady?

ScanReady is a Blender addon that transforms high-poly scans into optimized, game-ready assets through a fully automated workflow.

---

## ⚡ Core Workflow

1. Clean and prepare scan  
2. Optimize mesh  
3. Generate smart UVs  
4. Bake textures  

All in **one click**.
