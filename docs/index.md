# ScanReady

Convert high-poly scans into game-ready assets in one click.

## Quick Start

1. Select your high-poly scan.
2. Open the ScanReady panel.
3. Click **ONE CLICK BAKE**.
4. Wait for the final optimized asset.

## Main Features

- One Click workflow
- Smart UV
- Auto Cage
- Multi-material bake
- Memory-safe baking
- Preserve sharp details

## Workflow

ScanReady follows a simple 3-step workflow:

1. **Preview / Reduce**
2. **UV / Cage**
3. **Bake / Output**
