# Step 2 – UV / Cage

This step generates UVs and prepares the baking cage.

## Smart UV

Uses Blender Smart UV Project.

### Modes
- Balanced → default
- More aggressive → fewer islands
- More precise → better detail

## Cage

The cage controls projection during baking.

### Show Cage
Displays the cage mesh.

### Auto Cage Extrusion
Automatically calculates cage distance.

## Action

Click **Generate UVs** after creating the preview.
